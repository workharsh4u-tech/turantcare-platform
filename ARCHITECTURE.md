# TurantCare Platform Architecture

TurantCare Platform is built as a modular healthcare intelligence system consisting of:

- Patient Web Portal
- AI Report Interpretation Engine
- Consultation Workflow Layer
- Supabase Cloud Database
- Authentication & Authorization Layer
- Notification Engine
- Deployment via Vercel CI/CD

## Core Stack
- React + Vite
- TypeScript
- Tailwind UI
- Supabase Backend
- Vercel Deployment
